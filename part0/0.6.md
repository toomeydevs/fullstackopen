# Exercise 0.6: New Note in Single Page App Diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: User writes a new note

    Note right of browser: User clicks the Save button

    Note right of browser: JavaScript intercepts the form submission and prevents the page reload

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate server
    Note right of browser: The request body contains the new note as JSON
    Note right of server: Server saves the new note
    server-->>browser: HTTP 201 Created
    deactivate server

    Note right of browser: JavaScript adds the new note to the local notes array

    Note right of browser: Browser re-renders the notes list without reloading the page
```
